# HyperAdmin

A modern, Pydantic-native admin interface for FastAPI, powered by HTMX.

## Full Description

HyperAdmin is a next-generation admin interface for FastAPI applications. Inspired by Django's admin, but rebuilt from the ground up for the modern Python web stack.

Tired of writing boilerplate CRUD interfaces or fighting with complex frontend frameworks? HyperAdmin automatically generates a beautiful, fast, and highly interactive admin panel directly from your Pydantic models. It leverages the power of HTMX for dynamic UIs without writing a single line of JavaScript, Tailwind CSS for a sleek and modern design, and Pydantic as the single source of truth for your data models.

## Core Features

*   **Pydantic-Native**: Your Pydantic models are the source of truth. No more duplicate model definitions or complex form libraries.
*   **HTMX-Powered**: A fast, dynamic, and interactive UI with the simplicity of server-rendered HTML.
*   **Beautiful Out-of-the-Box**: Comes with a polished and modern UI built with Tailwind CSS.
*   **FastAPI Integration**: Designed to feel like a native part of FastAPI, with full support for async, dependency injection, and more.
*   **Zero Frontend Build Step**: Drop it in and go. No `npm install` or complex JavaScript toolchain needed.
*   **Easy to Customize**: Start with sensible defaults, then override and extend to fit your needs.
