## Stack Guidelines

### ✅ Use
- HTMX for server interactions ([examples](https://htmx.org/examples/))
- Alpine.js for client state
- Tailwind utilities for styling
- SQLModel with proper typing
- Pydantic for validation
- Flowbite UI components

### ❌ Avoid
- jQuery or Bootstrap classes
- WTForms for validation
- Inline JavaScript without Alpine.js
- Direct database queries in templates
- Missing type hints
